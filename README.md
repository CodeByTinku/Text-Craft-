## How to Use

1.  **Clone or Download** the repository to your local machine.
2.  Navigate to the project folder.
3.  Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).
4.  Start typing and formatting your text using the toolbar options!

## Project Structure

*   `index.html`: The main HTML file containing the editor's structure and toolbar.
*   `style.css`: The stylesheet defining the look and feel of the editor.
*   `script.js`: The JavaScript file containing the logic for text formatting and user interactions.
